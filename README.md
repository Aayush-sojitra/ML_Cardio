# CardioShield: Cardiovascular Disease Prediction AI

CardioShield is a predictive analytics platform that leverages machine learning to assess cardiovascular risk based on clinical parameters. The project uses a **Random Forest Classifier** trained on 70,000 patient records to provide high-precision diagnostics.

## 🚀 Key Features
- **Predictive Engine**: Real-time risk assessment using an optimized Random Forest model.
- **Clinical Dashboard**: Interactive visualization of medical data trends and model performance.
- **Responsive UI**: Modern, dark-themed interface built with React and Tailwind CSS.
- **Detailed Methodology**: Transparent view of data preprocessing, feature engineering, and validation metrics.

## 📊 Model Performance
- **Algorithm**: Random Forest (200 estimators, depth 20)
- **Accuracy**: 73.1% (aligned with clinical benchmarks)
- **Precision**: 0.74
- **Recall**: 0.70
- **F1 Score**: 0.72

## 🛠️ Technology Stack
- **Frontend**: React.js, Vite, Tailwind CSS, Lucide React, Recharts.
- **Backend**: Flask (Python), Scikit-learn, Pandas, NumPy.
- **DevOps**: Git, GitHub.

## ⚙️ Setup Instructions

### Backend
1. Navigate to the `backend` directory.
2. Install dependencies: `pip install flask flask-cors pandas scikit-learn`.
3. Start the API: `python app.py`.

### Frontend
1. Navigate to the `FrontendV` directory.
2. Install dependencies: `npm install`.
3. Start the dev server: `npm run dev`.
